# SN74LV1T34 | Single Buffer GATE COMOS Logic Level Shifter

The SN74LV1T34 is a logic level shifter with several options for both up and down translation.

Key Specs:
- Supports up to 50MHz
- 5V tolerance on pins
- Output Drive
    - 5.0V - 8mA
    - 3.3V - 7mA
    - 2.5V - 3mA
